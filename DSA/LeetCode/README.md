## 📅 Practice Tracks

### 1. Blind 75 (Must-Do)
The absolute minimum set of problems covering all key DSA patterns:
- **Arrays**: Two Sum, Best Time to Buy/Sell Stock, Product of Array Except Self.
- **Two Pointers**: Valid Palindrome, 3Sum, Container With Most Water.
- **Sliding Window**: Longest Substring Without Repeating Characters.
- **Linked Lists**: Reverse Linked List, Merge Two Sorted Lists, Reorder List.
- **Trees**: Maximum Depth of Binary Tree, Same Tree, Binary Tree Level Order Traversal.
- **Heaps**: Merge k Sorted Lists.
- **Dynamic Programming**: Climbing Stairs, Coin Change, Longest Common Subsequence.
- **Graphs**: Number of Islands, Clone Graph, Course Schedule.

### 2. NeetCode 150 (Comprehensive)
An expansion of Blind 75 detailing additional concepts like Trie, Backtracking, Advanced Graphs, and Bit Manipulation.

## 🔍 Best Solving Routine

1. **Understand Constraints**: Calculate the Big O limits of the problem.
2. **Brute Force First**: Speak or write the brute force $O(n^2)$ solution first to establish a baseline.
3. **Optimize Space/Time**: Think of sorting, hashing, sliding windows, or two pointers to reach $O(n \log n)$ or $O(n)$.
4. **Implement Cleanly**: Write modular, readable code, and check edge cases (empty inputs, negative bounds, overflow limits).

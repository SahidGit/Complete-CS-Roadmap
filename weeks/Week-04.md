# Week 04 — JavaScript Fundamentals

## Goal
Write modular browser programs that respond to user input and update the DOM safely.

## Learning Objectives
- Use values, control flow, functions, arrays and objects.
- Manipulate the DOM through events and render functions.
- Use ES modules and debug with browser developer tools.

## Topics and Concepts
Variables, types, equality, conditionals, loops, functions, scope, arrays, objects, destructuring, DOM selection, events, template literals, modules and debugging.

## Interview Questions
1. What is lexical scope?
2. When is event delegation useful?
3. What is the difference between `map`, `filter` and `reduce`?

## Mini Projects
1. **Calculator** — beginner, 6h. Arithmetic UI with keyboard support. *Skills:* state, events. *Extension:* add calculation history.
2. **Weather App** — intermediate, 8h. Weather interface with loading states. *Skills:* DOM rendering. *Extension:* remember locations.
3. **Todo App** — intermediate, 7h. Add, complete and filter tasks. *Skills:* array updates. *Extension:* add categories.

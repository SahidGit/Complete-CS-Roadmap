# Frontend Reference

Use [MDN Learn](https://developer.mozilla.org/en-US/docs/Learn_web_development) as the primary reference for HTML, CSS and browser JavaScript. Build accessible, keyboard-operable interfaces before adding frameworks.

## YouTube Channels

- **Sheryian's Coding School** – concise frontend tutorials.
- **Apna College** – latest frontend videos and project walkthroughs.
- **freeCodeCamp** – project ideas to build while learning.

### Suggested Tasks

- **Before starting frontend**: Set up a basic HTML page with proper doctype, meta charset, and viewport.
- **After finishing a concept**: Build a small project (e.g., a responsive navigation bar, a form with validation, a mini portfolio) and deploy to GitHub Pages.
